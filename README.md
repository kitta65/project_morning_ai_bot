## Q and A
### 何時に届くの？
アイちゃんの投稿が基本的に7時なので、まずは7時05分に投稿を検索しています。

何らかの理由でアイちゃんの投稿が遅れた場合、時間をおいて何回か再検索を試みる仕様です。

### アイコン画像がかわいくない
誰かがイラストを書いてくれたら変えます。Issuesからでも何でも連絡くだされば。

### 何か注意事項とかある？
このアカウントを友達登録してもTwitterのフォローは外さないでください。

YouTubeのチャンネル登録もお願いします。通知もONにしてね！
